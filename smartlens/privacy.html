سياسة الخصوصية

نحترم خصوصيتك في Smart Lens وملتزمون بحماية بياناتك الشخصية.

1. جمع البيانات

لا يتطلب التطبيق تسجيل دخول، ولا يجمع البريد الإلكتروني أو أي معرف شخصي.

يجمع التطبيق بيانات استخدام مؤقتة فقط مثل الصور والنصوص الممسوحة، ويتم معالجتها محليًا أو عبر خدمات موثوقة مثل Google Cloud Vision API وML Kit. هذه البيانات لا تُخزن بشكل دائم على الجهاز أو السيرفر.

2. الخدمات الخارجية

يستخدم التطبيق خدمات موثوقة لمعالجة الصور والنصوص:

  • Google Cloud Vision API
  • ML Kit
  • Firebase (Analytics, Crashlytics)

3. أمن البيانات ومشاركتها

البيانات مؤقتة وتُحذف فور مسح التطبيق أو إنهاء المستخدم لهويته الفريدة. لا تتم مشاركة البيانات مع أي طرف ثالث خارج نطاق الخدمات الموثوقة.

4. حقوق المستخدم

يمكن للمستخدم حذف التطبيق لإزالة البيانات، وستتم إعادة إنشاء هوية جديدة عند إعادة التثبيت.

للاستفسارات: support@brimagine.com

Privacy Policy

We respect your privacy at Smart Lens and are committed to protecting personal data.

1. Data Collection

The app does not require login and does not collect email or personal identifiers.

It only collects temporary usage data such as scanned images and text, processed locally or via trusted services like Google Cloud Vision API and ML Kit. No data is stored permanently on the device or server.

2. External Services

  • Google Cloud Vision API
  • ML Kit
  • Firebase (Analytics, Crashlytics)

3. Data Security & Sharing

Data is temporary and deleted when the app is uninstalled or user identity is reset. No data is shared with third parties outside trusted services.

4. User Rights

Users may remove data by uninstalling the app. A new identity is generated upon reinstall.
