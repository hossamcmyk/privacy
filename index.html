سياسة الخصوصية

نحن في Brimagine نحترم خصوصية المستخدمين ونلتزم بحماية بياناتهم الشخصية.

1. جمع البيانات

يقوم التطبيق بجمع بعض البيانات الأساسية مثل البريد الإلكتروني ومعرف المستخدم (UID) عند إنشاء الحساب.

وقد يتم جمع اسم المستخدم في حال إدخاله من قبل المستخدم داخل التطبيق.

قد يتم جمع بيانات استخدام عامة بهدف تحسين أداء التطبيق وتجربة المستخدم، دون جمع معلومات تعريفية مباشرة.

2. الخدمات الخارجية

يستخدم التطبيق خدمات خارجية موثوقة مثل Firebase Authentication و Firebase Analytics، والتي قد تقوم بمعالجة بعض البيانات وفقاً لسياسات الخصوصية الخاصة بها.

3. أمن البيانات ومشاركتها

يتم نقل وتخزين البيانات بشكل آمن باستخدام خدمات موثوقة مثل Firebase.

لا نقوم ببيع أو مشاركة بيانات المستخدمين مع أي أطراف خارجية خارج نطاق تشغيل الخدمة.

4. حقوق المستخدم

يمكن للمستخدم طلب حذف الحساب والبيانات المرتبطة به في أي وقت.

البريد الإلكتروني: support@brimagine.com

The app collects basic data such as Email and User ID (UID) during account creation.

The app uses trusted third-party services such as Firebase Authentication and Firebase Analytics, which may process certain data according to their own privacy policies.

3. Data Security and Sharing

Data is transmitted and stored securely using trusted services such as Firebase.

4. User Rights

Users may request deletion of their account and associated data at any time.
